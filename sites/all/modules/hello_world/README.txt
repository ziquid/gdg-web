Requirements:

Hello World Project

- The text "Hello World!" should appear in a bold typeface on the right
  sidebar of all Article pages only.
- When viewing an Article on the Drupal site, the phrase "Content starts here!"
  should appear in an italic typeface as the first line of content.
- These features should be portable, contained in a Drupal module file.
- Name the module "hello_world".
- Compatible with Drupal 7.
